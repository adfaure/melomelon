Getting started

Register the bot on twitch

First, register the bot using the following link. After this step, you should have an access token for the bot, keep it and don't communicate it to anyone. It will be used, to connect the bot to twitch. More information about bots here.

Connection parameters

Extension parameters

Comma separated list of valid commands prefixed by "!". For example !song,!musique,!zic. The extension needs to be reloaded (restart chrome).

Don't touch this parameters unless the bot is being disconnected during youtube playing. You can try to lower this number to maintain the connection active, note that it could also affect performances.

This parameter must be less than or equal to 295.
